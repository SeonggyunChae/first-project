인터넷의 시작

미국의 신기술을 연구하는 ARPA에서 현재 웹의 모태가 되는 아르파넷(ARPANET)을 1969년에 개발함
핵과 같은 공격에 대비해 전체 통신 시스템에서 데이터를 안전하게 보관 및 전송할 수 있는 시스템
우리나라는 1982년 한국전자기술연구소(ETRI)와 서울대학교 컴퓨터공학과가 인터넷 처음 만듦
1995년부터 전세계에 연결되고 일반인이 사용할 수 있었음

웹 브라우저 전쟁의 시작

NCSA는 1993년 모자이크라는 GUI 웹 브라우저 최초로 발표
네스케이프 커뮤니케이션(마크 안데르센)은 1994년 네스케이프 웹 브라우저 발표
마이크로소프트 인터넷 익스플로러 발표
초반 네스케이프 브라우저의 점유율이 높았으나, 마이크로소프트사에서 윈도우 운영체제에 익스플로러를 포함시키고
매킨토시의 기본 브라우저로 5년간 계약을 맺어 점유율을 높임

웹2.0 시대

서로 다른 사용자가 함께 새로운 콘텐츠를 창조할 수 있는 시대
유튜브, 위키피디아, 깃허브 등

웹 브라우저와 웹 표준

인터넷 익스플로러가 전 세계적으로 점유율을 높이면서 액티브X 같은 플러그인을 삽입해 웹 사이트가 점점 무거워지고 있음
2004년 W3C 회의에서 모질라 재단과 오페라 소프트웨어가 새로운 HTML 표준을 제안하지만 거절 당함
웹 브라우저 제공 기업(애플, 모질라, 오페라 소프트웨어)에서 2004년 6월 새로운 웹 표준 기관 WHATWG 설립
